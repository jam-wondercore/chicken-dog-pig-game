# 專案概述

## 專案名稱
Chicken-Dog-Pig Game - 節奏連拍遊戲

## 專案簡介
一款基於 React 的節奏連拍遊戲，讓使用者可以上傳自訂圖片，配合音樂節奏進行遊戲挑戰。

## 核心特色
- 自由新增/管理多個圖片組別
- 支援批次與單張圖片上傳
- 配合音樂節奏的連拍挑戰
- 簡潔易用的 Tab 切換介面

## 技術棧
- React 18+
- Vite (構建工具)
- Tailwind CSS (樣式框架)
- JavaScript/JSX

## 目標平台
PC 桌面版（Desktop Layout 優先）
