## 設計規範

### 色彩配置
```
主色（Primary）： #3B82F6  藍色
成功（Success）： #10B981  綠色
錯誤（Error）：   #EF4444  紅色
背景（Background）：#F9FAFB  淺灰
文字（Text）：     #1F2937  深灰
邊框（Border）：   #E5E7EB  中灰
```

### 間距系統
- 超小：4px  (space-1)
- 小：  8px  (space-2)
- 中：  16px (space-4)
- 大：  24px (space-6)
- 超大：32px (space-8)

### 組件樣式規範

#### 按鈕
```
- 圓角：rounded-lg (8px)
- 內距：px-6 py-3
- 字體：font-semibold
- 過渡：transition-all duration-200
- Hover：hover:opacity-90
```

#### 網格格子
```
- 尺寸：正方形（aspect-ratio: 1）
- 圓角：rounded-md (6px)
- 邊框：border-2
- 陰影：hover:shadow-lg
- 圖片：object-cover
```

#### Tab 標籤
```
- 未選中：bg-gray-200 text-gray-700
- 已選中：bg-blue-500 text-white
- 過渡：transition-colors duration-200
```

### 互動效果

#### Hover 效果
- 按鈕：hover:brightness-110
- 格子：hover:scale-105
- Tab：hover:bg-opacity-90

#### Active 效果
- **節奏外框**：當前應點擊的格子 - 粗外框高亮（border-4 border-yellow-400 或發光效果）
- **已點擊格子**：顯示✓標記或變暗（opacity-50）
- 正確點擊：短暫的綠色閃爍動畫
- 錯誤點擊：紅色閃爍 + 搖晃動畫

## 響應式設計（PC 優先）

### 容器最大寬度
- 設定頁容器：max-w-4xl (896px)
- 遊戲頁容器：max-w-3xl (768px)
- 網格容器：固定寬高比

### 最小支援寬度
- 最小寬度：1024px (桌面版)
